## Question - 1 :

> 1 2 3
> 1 2 3
> 1 2 3

### **`CPP :`**

**Using For-loops :**

```cpp
for (int i = 1; i <= 3; i++)
    {
        for (int j = 1; j <= 3; j++)
        {
            cout << j << " ";
        }
        cout << endl;
    }
```

**Using While-Loop :**

```cpp
int i = 1;
while (i <= n)
{
    int j = 1;
    while (j <= n)
    {
        cout << j << " ";
        j++;
    }
    cout << endl;
    i++;
}
```

## Question - 2 :

> 3 2 1
> 3 2 1
> 3 2 1

**Using While Loop :**

```cpp
int i = n;
while (i > 0)
{
    int j = n;
    while (j > 0)
    {
        cout << j << " ";
        j--;
    }
    cout << endl;
    i--;
}
```

## Question - 3 :

> 1
> 2 3
> 3 4 5
> 4 5 6 7

**Using While loop :**

```cpp
int i = 1;
while (i <= n)
{
    int j = 1;
    int count = i;
    while (j <= i)
    {
        cout << count << " ";
        count++;
        j++;
    }
    cout << endl;
    i++;
}
```

## Question - 5 :

```cpp

{
    int j = 1;
```
